# ⚽ W杯注目国・選手サイト　🔗 **[サイトを見る](https://worldcup-guide-2026-2yoo.vercel.app/)**

2026年FIFAワールドカップの注目国と注目選手をピックアップし紹介するWebアプリケーション



---

## 🎯 機能

### 1. トップページ `/`
- 全注目国をカード形式で表示
- 「国旗」「国名」「FIFAランク」をFIFAランク順で表示

### 2. 国別詳細ページ `/countries/[id]`
- 国の詳細情報表示
- チーム写真、国旗
- その国の注目選手一覧

### 3. 選手詳細ページ `/players/[id]`
- 選手の詳細情報表示
- 選手写真、名前、ポジション、所属クラブ
- 所属国へのリンク

---

## ℹ️ コンテンツ管理

### 新しい国を追加する場合
1. microCMS にログイン
2. コンテンツ「注目国」に新規追加

### 新しい選手を追加する場合
1. microCMS にログイン
2. コンテンツ「選手」に新規追加
